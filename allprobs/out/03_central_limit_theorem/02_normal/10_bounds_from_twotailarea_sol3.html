First, draw a sketch. We can label the \(F\) axis by adding integer multiples of 0.5 to 2.3. We know to shade the the outsides because values far from 2.3 satisfy the condition \(\big|F-2.3\big|>d\) (regardless of the exact value of \(d\)). We don’t know exactly where to place the boundaries, but we know the two-tail area is 0.56.


As an intermediate step, let’s find \(z\) such that \(P(|Z|>z)=0.56\). First, we need to evaluate \(P(Z<z)\).

\[\begin{aligned} P(Z<z) &= \frac{2-P(|Z|>z)}{2} \\ \\ &= \frac{2-0.56}{2} \\ \\ &= 0.72 \end{aligned}\]

You could have also drawn some pictures… we know there is symmetry and all the areas should add to 1.



We find \(z\) such that \(P(Z<z)=0.72\).

By using the \(z\) table we can determine \(z\).

\(z\) \(P(Z<z)\)
0.56 0.7123
0.57 0.7157
0.58 0.719
0.59 0.7224
0.6 0.7257
0.57 0.7291

Either 0.58 or 0.59 is a good estimation of \(z\), and either value will lead you to an acceptable answer. Using other tools, a more accurate value can be found. I will show the work with a more accurate value. \[z=0.5828\]

We now convert the \(z\) score into a \(d\) score. \[\begin{aligned} d &= z\sigma \\ &= (0.5828)(0.5) \\ &= 0.2914 \end{aligned}\]

We can also visualize this with a spinner.

  2. It is easy to find \(f_1 = 2.3-0.2914 = 2.0086\)
  3. It is easy to find \(f_2 = 2.3+0.2914 = 2.5914\)
