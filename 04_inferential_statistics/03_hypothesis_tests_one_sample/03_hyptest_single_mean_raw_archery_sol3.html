Determine the sample statistics (\(\bar{x}\) and \(s\)). I would recommend using a spreadsheet or R (or another computer-based method). \[\bar{x} = 7.65\] \[s = 38.46 \] Evaluate the \(t\) statistic.

\[t = \frac{|7.65-0|}{(38.46)/\sqrt{60}} = 1.54 \] Restate the \(p\)-value. \[p\text{-value} = P\left(|T|>1.54\right) \] Remember, the degree of freedom is one less than the sample size. \[\text{df} = 59 \]

At this point, there are various ways to determine the \(p\)-value. The least accurate way is to use the \(t\)-table. Go to the row with \(\text{df}=n-1=60-1=59\).


If we go to the row with df=59, we can see that our calculated \(t\), 1.54, is between 1.3 and 1.67. Thus, we know \(P(|T|>1.54)\) is between 0.1 and 0.2. You could get pretty close by using linear interpolation. \[P(|T|>1.54)\approx \left(\frac{1.54-1.3}{1.67-1.3}\right)\left(0.1-0.2\right)+0.2 = 0.135\]

The more accurate \(p\)-value is 0.1287035. This can be calculated using a computer; for example, you can use this web app (for full accuracy, youâ€™ll need the more precise value of \(t=1.5408357\)). You can also use a spreadsheet or R.
