First, determine the interval of typical measurements.

\[\begin{aligned} \text{interval of typical measurements} &= (\mu-2\sigma,\,\mu+2\sigma)\\ &= (31,39) \end{aligned}\]

Now, determine how many measurements (and divide by 200 for what proportion) are either less than 31 or more than 39. You’ll want to use a computer.

R

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum(x<31 | x>39)
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.035

In R, the “|” operator means “or”.

You could also write the equality as an absolute deviation from the mean. Any measurement more than 4 units from 35 (in either direction) would be outside the interval.

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum( abs(x-35)>4 )
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.035

Spreadsheet

In a spreadsheet you can use the IF function along with the OR function to determine which measurements are under 31 or over 39. You then use the SUM function to count the 1s.
