A sample’s statistic typically approaches its corresponding population parameter, but with a small sample size there is regularly error. Let’s explore this idea with an example.

A geometric distribution is a well-studied discrete population. The spinner below represents a geometric distribution with the following population parameters. \[\mu = 3.21\] \[\sigma = 3.68\]


That spinner was spun many times. The raw data is displayed below and can be downloaded as a CSV.

1, 1, 0, 1, 7, 8, 1, 5, 1, 1, 0, 9, 6, 5, 1, 1, 10, 11, 7, 4, 0, 10, 2, 0, 0, 0, 0, 1, 1, 14, 3, 3, 1, 0, 10, 2, 1, 3, 6, 3, 4, 1, 8, 4, 0, 0, 1, 2

Please calculate the following sample statistics.


  1. What is the sample size? (Determine \(n\).)
  2. What is the sample mean? (Determine \(\bar{x}\). You can round to nearest hundredth.)
  3. What is the absolute difference between the sample mean and population mean? (Calculate \(|\bar{x}-\mu|\). You can round to nearest hundredth.)
  4. What is the sample median?
  5. What is the sample variance? (Determine \(s^2\). Use Bessel’s correction. You can round to nearest hundredth.)
  6. What is the sample standard deviation? (Determine \(s\). Use Bessel’s correction. You can round to nearest hundredth.)
  7. What is the absolute difference between the sample standard deviation and population standard deviation? (Calculate \(|s-\sigma|\). You can round to nearest hundredth.)
