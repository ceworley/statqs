First, draw a sketch. We can label the \(B\) axis by adding integer multiples of 5 to 21. We know to shade toward the left because small values of \(B\) satisfy the condition \(B<19\).


We are given a specific \(b\) value as a boundary. (Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) \[b = 19 \] We calculate the \(z\) value of the boundary. \[\begin{aligned} z &= \frac{b-\mu}{\sigma} \\\\ &= \frac{19-21}{5} \\\\ &= -0.4 \end{aligned}\]

We have rephrased our problem into a standard normal probability problem, because \[P(B<19) ~=~ P(Z<-0.4) \]

So, we just need to evaluate \(P(Z<-0.4)\). To do this, you just need a \(z\)-table.

\(z\) \(P(Z<z)\)
-0.42 0.3372
-0.41 0.3409
-0.4 0.3446
-0.39 0.3483
-0.38 0.352

Thus, we find our answer. \[P(B<19) = 0.3446 \]
