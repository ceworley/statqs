To do this by hand, you first calculate the sample statistics. \[n_1 = 40\] \[n_2 = 30\] \[\bar{x}_1 = 497.7\] \[\bar{x}_2 = 513\] \[s_1 = 34\] \[s_2 = 32.5\] \[|t| = 1.9\] \[\text{df} = 64.007\] You will get quite close if you round df down (floor).

Then, using a computer application or a \(t\) table, you should be able to determine the following probabilities (using interpolation to estimate if using \(t\) table). \[P(T<1.9) = 0.9693 \] \[P(|T|>1.9) = 0.0614 \] \[p\text{-value} = 0.0614 \]

read.csv

Download triglyceride.csv and move it to a directory (folder). Make a script, welchttest.r, and put it in the same directory. Set the working directory to this directory. Then, run the script.

### welchttest.r
data = read.csv("triglyceride.csv")
x1 = data$x1
x2 = data$x2
t.test(x1,x2)
