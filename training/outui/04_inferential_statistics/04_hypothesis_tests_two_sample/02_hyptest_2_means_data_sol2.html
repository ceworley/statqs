To do this by hand, you first calculate the sample statistics. \[n_1 = 23\] \[n_2 = 39\] \[\bar{x}_1 = 382.2\] \[\bar{x}_2 = 366\] \[s_1 = 34.1\] \[s_2 = 54.8\] \[|t| = 1.44\] \[\text{df} = 59.764\] You will get quite close if you round df down (floor).

Then, using a computer application or a \(t\) table, you should be able to determine the following probabilities (using interpolation to estimate if using \(t\) table). \[P(T<1.44) = 0.922 \] \[P(|T|>1.44) = 0.1561 \] \[p\text{-value} = 0.1561 \]

read.csv

Download triglyceride.csv and move it to a directory (folder). Make a script, welchttest.r, and put it in the same directory. Set the working directory to this directory. Then, run the script.

### welchttest.r
data = read.csv("triglyceride.csv")
x1 = data$x1
x2 = data$x2
t.test(x1,x2)
