First, determine \(z^\star\) such that \(P(|Z|<z^\star)=0.67\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.67}{2} = 0.165\]


Determine a leftward area with boundary \(z^\star\). \[P(Z<z^\star)=\gamma + \frac{1-\gamma}{2}= 0.67+0.165=0.835\] It should be mentioned the expression can be simplified. \[P(Z<z^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.335=0.835\]


We rephrase the puzzle. We wish to determine \(z^\star\) such that \(P(Z<z^\star)=0.835\). This is easy to determine using a \(z\)-table, qnorm in R, norm.inv in spreadsheets, or other methods for evaluating the quantile function.

\[P(Z<0.97)=0.835\] In the original phrasing,

\[P(|Z|<0.97)=0.67\]

So, \(z^\star = 0.97\). Now, use the given expressions.

\[\text{Lower bound} = 52.9-(0.97)(13.4)/\sqrt{180} ~=~ 51.9\] \[\text{Upper bound} = 52.9+(0.97)(13.4)/\sqrt{180} ~=~ 53.9\]

Spreadsheet

A spreadsheet can use the confidence.norm function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.67=0.33\)
  • Population standard deviation, \(\sigma=13.4\)
  • Sample size, \(n=180\)

The function returns the margin of error. \[\text{margin of error} = \frac{z^\star \sigma}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.norm(1-0.67,13.4,180), the result would be the margin of error, 0.9729226. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 52.9-0.973 = 51.927\] \[\bar{x}+\text{ME} = 52.9+0.973 = 53.873\]

You can do this with a spreadsheet.


R

xbar = 52.9
sigma = 13.4
n = 180
gamma = 0.67
zstar = qnorm(gamma+(1-gamma)/2)
ME = zstar*sigma/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
