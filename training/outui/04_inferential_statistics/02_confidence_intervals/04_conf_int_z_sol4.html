First, determine \(z^\star\) such that \(P(|Z|<z^\star)=0.98\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.98}{2} = 0.01\]


Determine a leftward area with boundary \(z^\star\). \[P(Z<z^\star)=\gamma + \frac{1-\gamma}{2}= 0.98+0.01=0.99\] It should be mentioned the expression can be simplified. \[P(Z<z^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.49=0.99\]


We rephrase the puzzle. We wish to determine \(z^\star\) such that \(P(Z<z^\star)=0.99\). This is easy to determine using a \(z\)-table, qnorm in R, norm.inv in spreadsheets, or other methods for evaluating the quantile function.

\[P(Z<2.33)=0.99\] In the original phrasing,

\[P(|Z|<2.33)=0.98\]

So, \(z^\star = 2.33\). Now, use the given expressions.

\[\text{Lower bound} = 92.1-(2.33)(27.8)/\sqrt{253} ~=~ 88\] \[\text{Upper bound} = 92.1+(2.33)(27.8)/\sqrt{253} ~=~ 96.2\]

Spreadsheet

A spreadsheet can use the confidence.norm function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.98=0.02\)
  • Population standard deviation, \(\sigma=27.8\)
  • Sample size, \(n=253\)

The function returns the margin of error. \[\text{margin of error} = \frac{z^\star \sigma}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.norm(1-0.98,27.8,253), the result would be the margin of error, 4.0659234. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 92.1-4.066 = 88.034\] \[\bar{x}+\text{ME} = 92.1+4.066 = 96.166\]

You can do this with a spreadsheet.


R

xbar = 92.1
sigma = 27.8
n = 253
gamma = 0.98
zstar = qnorm(gamma+(1-gamma)/2)
ME = zstar*sigma/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
