Determine the sample statistics (\(\bar{x}\) and \(s\)). I would recommend using a spreadsheet or R (or another computer-based method). \[\bar{x} = 11.3\] \[s = 31.12 \] Evaluate the \(t\) statistic.

\[t = \frac{|11.3-0|}{(31.12)/\sqrt{60}} = 2.81 \] Restate the \(p\)-value. \[p\text{-value} = P\left(|T|>2.81\right) \] Remember, the degree of freedom is one less than the sample size. \[\text{df} = 59 \]

At this point, there are various ways to determine the \(p\)-value. The least accurate way is to use the \(t\)-table. Go to the row with \(\text{df}=n-1=60-1=59\).


If we go to the row with df=59, we can see that our calculated \(t\), 2.81, is between 2.66 and 2.92. Thus, we know \(P(|T|>2.81)\) is between 0.005 and 0.01. You could get pretty close by using linear interpolation. \[P(|T|>2.81)\approx \left(\frac{2.81-2.66}{2.92-2.66}\right)\left(0.005-0.01\right)+0.01 = 0.00712\]

The more accurate \(p\)-value is 0.0066653. This can be calculated using a computer; for example, you can use this web app (for full accuracy, you’ll need the more precise value of \(t=2.8124996\)). You can also use a spreadsheet or R.
