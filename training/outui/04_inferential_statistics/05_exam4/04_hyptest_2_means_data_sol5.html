Then, using a computer application or a \(t\) table, you should be able to determine the following probabilities (using interpolation to estimate if using \(t\) table). \[P(T<1.42) = 0.9126 \] \[P(|T|>1.42) = 0.1749 \] \[p\text{-value} = 0.1749 \]

The \(p\)-value is more than 0.05, so the result is NOT significant.

read.csv

Download triglyceride.csv and move it to a directory (folder). Make a script, welchttest.r, and put it in the same directory. Set the working directory to this directory. Then, run the script.
