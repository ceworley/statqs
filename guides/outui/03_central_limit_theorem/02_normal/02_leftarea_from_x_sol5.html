First, draw a sketch. We can label the \(V\) axis by adding integer multiples of 16 to 78. We know to shade toward the left because small values of \(V\) satisfy the condition \(V<74\).


We are given a specific \(v\) value as a boundary. (Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) \[v = 74 \] We calculate the \(z\) value of the boundary. \[\begin{aligned} z &= \frac{v-\mu}{\sigma} \\\\ &= \frac{74-78}{16} \\\\ &= -0.25 \end{aligned}\]

We have rephrased our problem into a standard normal probability problem, because \[P(V<74) ~=~ P(Z<-0.25) \]

So, we just need to evaluate \(P(Z<-0.25)\). To do this, you just need a \(z\)-table.

\(z\) \(P(Z<z)\)
-0.27 0.3936
-0.26 0.3974
-0.25 0.4013
-0.24 0.4052
-0.23 0.409

Thus, we find our answer. \[P(V<74) = 0.4013 \]
