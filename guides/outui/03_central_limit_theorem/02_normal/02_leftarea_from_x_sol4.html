First, draw a sketch. We can label the \(C\) axis by adding integer multiples of 50 to 190. We know to shade toward the left because small values of \(C\) satisfy the condition \(C<220\).


We are given a specific \(c\) value as a boundary. (Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) \[c = 220 \] We calculate the \(z\) value of the boundary. \[\begin{aligned} z &= \frac{c-\mu}{\sigma} \\\\ &= \frac{220-190}{50} \\\\ &= 0.6 \end{aligned}\]

We have rephrased our problem into a standard normal probability problem, because \[P(C<220) ~=~ P(Z<0.6) \]

So, we just need to evaluate \(P(Z<0.6)\). To do this, you just need a \(z\)-table.

\(z\) \(P(Z<z)\)
0.58 0.719
0.59 0.7224
0.6 0.7257
0.61 0.7291
0.62 0.7324

Thus, we find our answer. \[P(C<220) = 0.7257 \]
