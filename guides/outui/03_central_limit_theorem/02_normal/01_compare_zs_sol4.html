The formula to determine the \(z\)-score of a measurement is the ratio with numerator the difference between measurement and population mean and denominator the population standard deviation.

\[z = \frac{x-\mu}{\sigma} \]

The highest \(z\)-score (furthest right on number line) corresponds to the most unusually large measurement.

The smallest \(z\)-score (furthest left on number line) corresponds to the most unusually small measurement.

The smallest absolute \(z\)-score corresponds to the most usually sized measurement.

The largest absolute \(z\)-score corresponds to the most unusually sized measurement.


  1. We use the formula for calculating the banana specimen’s \(z\) score. \[z = \frac{255.2-212}{24} = 1.8 \]
  2. Use the \(z\) table to find the cumulative probability \[P(Z<1.8) = 0.9641 \]
  3. We use the formula for calculating the mango specimen’s \(z\) score. \[z = \frac{232.8-245}{36} = -0.34 \]
  4. Use the \(z\) table to find the cumulative probability \[P(Z<-0.34) = 0.3669 \]
  5. We use the formula for calculating the watermelon specimen’s \(z\) score. \[z = \frac{7412-7697}{500} = -0.57 \]
  6. Use the \(z\) table to find the cumulative probability \[P(Z<-0.57) = 0.2843 \]
  7. We use the formula for calculating the coconut specimen’s \(z\) score. \[z = \frac{673.2-724}{62} = -0.82 \]
  8. Use the \(z\) table to find the cumulative probability \[P(Z<-0.82) = 0.2061 \]
