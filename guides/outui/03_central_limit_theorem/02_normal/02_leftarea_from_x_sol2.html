First, draw a sketch. We can label the \(I\) axis by adding integer multiples of 50 to 790. We know to shade toward the left because small values of \(I\) satisfy the condition \(I<780\).


We are given a specific \(i\) value as a boundary. (Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) \[i = 780 \] We calculate the \(z\) value of the boundary. \[\begin{aligned} z &= \frac{i-\mu}{\sigma} \\\\ &= \frac{780-790}{50} \\\\ &= -0.2 \end{aligned}\]

We have rephrased our problem into a standard normal probability problem, because \[P(I<780) ~=~ P(Z<-0.2) \]

So, we just need to evaluate \(P(Z<-0.2)\). To do this, you just need a \(z\)-table.

\(z\) \(P(Z<z)\)
-0.22 0.4129
-0.21 0.4168
-0.2 0.4207
-0.19 0.4247
-0.18 0.4286

Thus, we find our answer. \[P(I<780) = 0.4207 \]
