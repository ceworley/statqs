Determine the sample statistics (\(\bar{x}\) and \(s\)). I would recommend using a spreadsheet or R (or another computer-based method). \[\bar{x} = -39.93\] \[s = 136.1 \] Evaluate the \(t\) statistic.

\[t = \frac{|-39.93-0|}{(136.1)/\sqrt{30}} = 1.61 \] Restate the \(p\)-value. \[p\text{-value} = P\left(|T|>1.61\right) \] Remember, the degree of freedom is one less than the sample size. \[\text{df} = 29 \]

At this point, there are various ways to determine the \(p\)-value. The least accurate way is to use the \(t\)-table. Go to the row with \(\text{df}=n-1=30-1=29\).


If we go to the row with df=29, we can see that our calculated \(t\), 1.61, is between 1.31 and 1.7. Thus, we know \(P(|T|>1.61)\) is between 0.1 and 0.2. You could get pretty close by using linear interpolation. \[P(|T|>1.61)\approx \left(\frac{1.61-1.31}{1.7-1.31}\right)\left(0.1-0.2\right)+0.2 = 0.123\]

The more accurate \(p\)-value is 0.1188151. This can be calculated using a computer; for example, you can use this web app (for full accuracy, you’ll need the more precise value of \(t=1.6073383\)). You can also use a spreadsheet or R.
