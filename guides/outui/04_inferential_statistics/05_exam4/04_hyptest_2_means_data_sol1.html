To do this by hand, you first calculate the sample statistics. \[n_1 = 28\] \[n_2 = 31\] \[\bar{x}_1 = 287.3\] \[\bar{x}_2 = 272.8\] \[s_1 = 20.1\] \[s_2 = 20.4\] \[|t| = 2.75\] \[\text{df} = 56.573\] You will get quite close if you round df down (floor).

Then, using a computer application or a \(t\) table, you should be able to determine the following probabilities (using interpolation to estimate if using \(t\) table). \[P(T<2.75) = 0.996 \] \[P(|T|>2.75) = 0.0079 \] \[p\text{-value} = 0.0079 \]

The \(p\)-value is less than 0.05, so the result is significant.

read.csv

Download triglyceride.csv and move it to a directory (folder). Make a script, welchttest.r, and put it in the same directory. Set the working directory to this directory. Then, run the script.
