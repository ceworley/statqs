To do this by hand, you first calculate the sample statistics. \[n_1 = 38\] \[n_2 = 19\] \[\bar{x}_1 = 247.2\] \[\bar{x}_2 = 281.6\] \[s_1 = 47.7\] \[s_2 = 52.3\] \[|t| = 2.41\] \[\text{df} = 33.298\] You will get quite close if you round df down (floor).

Then, using a computer application or a \(t\) table, you should be able to determine the following probabilities (using interpolation to estimate if using \(t\) table). \[P(T<2.41) = 0.9893 \] \[P(|T|>2.41) = 0.0214 \] \[p\text{-value} = 0.0214 \]

The \(p\)-value is less than 0.05, so the result is significant.
