To do this by hand, you first calculate the sample statistics. \[n_1 = 24\] \[n_2 = 20\] \[\bar{x}_1 = 405\] \[\bar{x}_2 = 368.3\] \[s_1 = 57.7\] \[s_2 = 46.1\] \[|t| = 2.34\] \[\text{df} = 41.943\] You will get quite close if you round df down (floor).

Then, using a computer application or a \(t\) table, you should be able to determine the following probabilities (using interpolation to estimate if using \(t\) table). \[P(T<2.34) = 0.9881 \] \[P(|T|>2.34) = 0.0239 \] \[p\text{-value} = 0.0239 \]

The \(p\)-value is less than 0.05, so the result is significant.
