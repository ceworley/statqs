First, determine \(t^\star\) such that \(P(|T|<t^\star)=0.96\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.96}{2} = 0.02\]


Thus, we determine a left area. \[P(T<t^\star)=\gamma + \frac{1-\gamma}{2}= 0.96+0.02=0.98\] It should be mentioned the expression can be simplified. \[P(T<t^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.48=0.98\]


We rephrase the puzzle. We wish to determine \(t^\star\) such that \(P(T<t^\star)=0.98\). This is easy to determine using a \(t\)-table, qt in R, t.inv in spreadsheets, or other methods for evaluating the quantile function.

Because \(n=36\), then \(\text{df}=35\).

\[P(T<2.13)=0.98\] In the original phrasing,

\[P(|T|<2.13)=0.96\]

So, \(t^\star = 2.13\). Now, use the given expressions.

\[\text{Lower bound} = 27.9-(2.13)(5.6)/\sqrt{36} ~=~ 25.9\] \[\text{Upper bound} = 27.9+(2.13)(5.6)/\sqrt{36} ~=~ 29.9\]

Spreadsheet

A spreadsheet can use the confidence.t function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.96=0.04\)
  • sample standard deviation, \(s=5.6\)
  • Sample size, \(n=36\)

The function returns the margin of error. \[\text{margin of error} = \frac{t^\star s}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.t(1-0.96,5.6,36), the result would be the margin of error, 1.9909462. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 27.9-1.991 = 25.909\] \[\bar{x}+\text{ME} = 27.9+1.991 = 29.891\]

You can do this with a spreadsheet.


R

xbar = 27.9
s = 5.6
n = 36
gamma = 0.96
df = n-1   # df is the degrees of freedom
tstar = qt(0.5+gamma/2, df)
ME = tstar*s/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
## The lower bound: 25.9091
## The upper bound: 29.8909
