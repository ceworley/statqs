First, determine \(z^\star\) such that \(P(|Z|<z^\star)=0.63\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.63}{2} = 0.185\]


Determine a leftward area with boundary \(z^\star\). \[P(Z<z^\star)=\gamma + \frac{1-\gamma}{2}= 0.63+0.185=0.815\] It should be mentioned the expression can be simplified. \[P(Z<z^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.315=0.815\]


We rephrase the puzzle. We wish to determine \(z^\star\) such that \(P(Z<z^\star)=0.815\). This is easy to determine using a \(z\)-table, qnorm in R, norm.inv in spreadsheets, or other methods for evaluating the quantile function.

\[P(Z<0.9)=0.815\] In the original phrasing,

\[P(|Z|<0.9)=0.63\]

So, \(z^\star = 0.9\). Now, use the given expressions.

\[\text{Lower bound} = 14.8-(0.9)(4.5)/\sqrt{230} ~=~ 14.5\] \[\text{Upper bound} = 14.8+(0.9)(4.5)/\sqrt{230} ~=~ 15.1\]

Spreadsheet

A spreadsheet can use the confidence.norm function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.63=0.37\)
  • Population standard deviation, \(\sigma=4.5\)
  • Sample size, \(n=230\)

The function returns the margin of error. \[\text{margin of error} = \frac{z^\star \sigma}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.norm(1-0.63,4.5,230), the result would be the margin of error, 0.2660027. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 14.8-0.266 = 14.534\] \[\bar{x}+\text{ME} = 14.8+0.266 = 15.066\]

You can do this with a spreadsheet.


R

xbar = 14.8
sigma = 4.5
n = 230
gamma = 0.63
zstar = qnorm(gamma+(1-gamma)/2)
ME = zstar*sigma/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
