First, determine \(t^\star\) such that \(P(|T|<t^\star)=0.98\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.98}{2} = 0.01\]


Thus, we determine a left area. \[P(T<t^\star)=\gamma + \frac{1-\gamma}{2}= 0.98+0.01=0.99\] It should be mentioned the expression can be simplified. \[P(T<t^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.49=0.99\]


We rephrase the puzzle. We wish to determine \(t^\star\) such that \(P(T<t^\star)=0.99\). This is easy to determine using a \(t\)-table, qt in R, t.inv in spreadsheets, or other methods for evaluating the quantile function.

Because \(n=26\), then \(\text{df}=25\).

\[P(T<2.49)=0.99\] In the original phrasing,

\[P(|T|<2.49)=0.98\]

So, \(t^\star = 2.49\). Now, use the given expressions.

\[\text{Lower bound} = 71.3-(2.49)(22.6)/\sqrt{26} ~=~ 60.3\] \[\text{Upper bound} = 71.3+(2.49)(22.6)/\sqrt{26} ~=~ 82.3\]

Spreadsheet

A spreadsheet can use the confidence.t function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.98=0.02\)
  • sample standard deviation, \(s=22.6\)
  • Sample size, \(n=26\)

The function returns the margin of error. \[\text{margin of error} = \frac{t^\star s}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.t(1-0.98,22.6,26), the result would be the margin of error, 11.0145533. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 71.3-11.015 = 60.285\] \[\bar{x}+\text{ME} = 71.3+11.015 = 82.315\]

You can do this with a spreadsheet.


R

xbar = 71.3
s = 22.6
n = 26
gamma = 0.98
df = n-1   # df is the degrees of freedom
tstar = qt(0.5+gamma/2, df)
ME = tstar*s/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
## The lower bound: 60.2854
## The upper bound: 82.3146
