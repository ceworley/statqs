First, determine \(z^\star\) such that \(P(|Z|<z^\star)=0.7\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.7}{2} = 0.15\]


Determine a leftward area with boundary \(z^\star\). \[P(Z<z^\star)=\gamma + \frac{1-\gamma}{2}= 0.7+0.15=0.85\] It should be mentioned the expression can be simplified. \[P(Z<z^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.35=0.85\]


We rephrase the puzzle. We wish to determine \(z^\star\) such that \(P(Z<z^\star)=0.85\). This is easy to determine using a \(z\)-table, qnorm in R, norm.inv in spreadsheets, or other methods for evaluating the quantile function.

\[P(Z<1.04)=0.85\] In the original phrasing,

\[P(|Z|<1.04)=0.7\]

So, \(z^\star = 1.04\). Now, use the given expressions.

\[\text{Lower bound} = 94.6-(1.04)(15.9)/\sqrt{116} ~=~ 93.1\] \[\text{Upper bound} = 94.6+(1.04)(15.9)/\sqrt{116} ~=~ 96.1\]

Spreadsheet

A spreadsheet can use the confidence.norm function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.7=0.3\)
  • Population standard deviation, \(\sigma=15.9\)
  • Sample size, \(n=116\)

The function returns the margin of error. \[\text{margin of error} = \frac{z^\star \sigma}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.norm(1-0.7,15.9,116), the result would be the margin of error, 1.5300637. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 94.6-1.53 = 93.07\] \[\bar{x}+\text{ME} = 94.6+1.53 = 96.13\]

You can do this with a spreadsheet.


R

xbar = 94.6
sigma = 15.9
n = 116
gamma = 0.7
zstar = qnorm(gamma+(1-gamma)/2)
ME = zstar*sigma/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
