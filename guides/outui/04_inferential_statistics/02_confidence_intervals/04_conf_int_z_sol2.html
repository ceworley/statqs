First, determine \(z^\star\) such that \(P(|Z|<z^\star)=0.65\). It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). \[\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.65}{2} = 0.175\]


Determine a leftward area with boundary \(z^\star\). \[P(Z<z^\star)=\gamma + \frac{1-\gamma}{2}= 0.65+0.175=0.825\] It should be mentioned the expression can be simplified. \[P(Z<z^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.325=0.825\]


We rephrase the puzzle. We wish to determine \(z^\star\) such that \(P(Z<z^\star)=0.825\). This is easy to determine using a \(z\)-table, qnorm in R, norm.inv in spreadsheets, or other methods for evaluating the quantile function.

\[P(Z<0.93)=0.825\] In the original phrasing,

\[P(|Z|<0.93)=0.65\]

So, \(z^\star = 0.93\). Now, use the given expressions.

\[\text{Lower bound} = 85.1-(0.93)(24.8)/\sqrt{88} ~=~ 82.6\] \[\text{Upper bound} = 85.1+(0.93)(24.8)/\sqrt{88} ~=~ 87.6\]

Spreadsheet

A spreadsheet can use the confidence.norm function. It takes three arguments:

  • Alpha (\(\alpha\)) is the two-tail area, \(\alpha=1-\gamma=1-0.65=0.35\)
  • Population standard deviation, \(\sigma=24.8\)
  • Sample size, \(n=88\)

The function returns the margin of error. \[\text{margin of error} = \frac{z^\star \sigma}{\sqrt{n}}\]

So, in this case, if in a spreadsheet you typed =confidence.norm(1-0.65,24.8,88), the result would be the margin of error, 2.4707633. You then add/subtract the margin of error to/from the sample mean. \[\bar{x}-\text{ME} = 85.1-2.471 = 82.629\] \[\bar{x}+\text{ME} = 85.1+2.471 = 87.571\]

You can do this with a spreadsheet.


R

xbar = 85.1
sigma = 24.8
n = 88
gamma = 0.65
zstar = qnorm(gamma+(1-gamma)/2)
ME = zstar*sigma/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
