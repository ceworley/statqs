To do this by hand, you first calculate the sample statistics. \[n_1 = 30\] \[n_2 = 17\] \[\bar{x}_1 = 256.9\] \[\bar{x}_2 = 243.8\] \[s_1 = 31.4\] \[s_2 = 19.4\] \[|t| = 1.76\] \[\text{df} = 44.558\] You will get quite close if you round df down (floor).

Then, using a computer application or a \(t\) table, you should be able to determine the following probabilities (using interpolation to estimate if using \(t\) table). \[P(T<1.76) = 0.9575 \] \[P(|T|>1.76) = 0.0849 \] \[p\text{-value} = 0.0849 \]

read.csv

Download triglyceride.csv and move it to a directory (folder). Make a script, welchttest.r, and put it in the same directory. Set the working directory to this directory. Then, run the script.

### welchttest.r
data = read.csv("triglyceride.csv")
x1 = data$x1
x2 = data$x2
t.test(x1,x2)
##
