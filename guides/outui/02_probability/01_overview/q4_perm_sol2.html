This scenario describes a permutations problem (order matters). We are considering the nonrepeating sequences of size 3 from a set of size 21.

\[\begin{aligned} {_nP_r} &= \frac{n!}{(n-r)!} \\\\ n &= 21 \\\\ r &= 3 \\\\ {_{21}P_{3}} &= \frac{21!}{(21-3)! } \\\\ &= \frac{21!}{18!} \\\\ &= 21\cdot20\cdot19 \\\\ &= 7980 \end{aligned}\]
