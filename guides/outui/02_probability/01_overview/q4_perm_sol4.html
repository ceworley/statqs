This scenario describes a permutations problem (order matters). We are considering the nonrepeating sequences of size 2 from a set of size 22.

\[\begin{aligned} {_nP_r} &= \frac{n!}{(n-r)!} \\\\ n &= 22 \\\\ r &= 2 \\\\ {_{22}P_{2}} &= \frac{22!}{(22-2)! } \\\\ &= \frac{22!}{20!} \\\\ &= 22\cdot21 \\\\ &= 462 \end{aligned}\]
