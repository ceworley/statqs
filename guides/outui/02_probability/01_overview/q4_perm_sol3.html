This scenario describes a permutations problem (order matters). We are considering the nonrepeating sequences of size 2 from a set of size 15.

\[\begin{aligned} {_nP_r} &= \frac{n!}{(n-r)!} \\\\ n &= 15 \\\\ r &= 2 \\\\ {_{15}P_{2}} &= \frac{15!}{(15-2)! } \\\\ &= \frac{15!}{13!} \\\\ &= 15\cdot14 \\\\ &= 210 \end{aligned}\]
