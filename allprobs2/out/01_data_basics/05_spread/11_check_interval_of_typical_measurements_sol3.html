First, determine the interval of typical measurements.

interval of typical measurements=(μ−2σ,μ+2σ)=(46,54)\begin{aligned} \text{interval of typical measurements} &= (\mu-2\sigma,\,\mu+2\sigma)\\ &= (46,54) \end{aligned}

Now, determine how many measurements (and divide by 200 for what proportion) are either less than 46 or more than 54. You’ll want to use a computer.

R

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum(x<46 | x>54)
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.035

In R, the “|” operator means “or”.

You could also write the equality as an absolute deviation from the mean. Any measurement more than 4 units from 50 (in either direction) would be outside the interval.

x = read.csv("check_interval_typical_measurements.csv")$x
n = length(x)
count_outside = sum( abs(x-50)>4 )
prop_outside = count_outside/n
print(prop_outside)
## [1] 0.035

Spreadsheet

In a spreadsheet you can use the IF function along with the OR function to determine which measurements are under 46 or over 54. You then use the SUM function to count the 1s.
