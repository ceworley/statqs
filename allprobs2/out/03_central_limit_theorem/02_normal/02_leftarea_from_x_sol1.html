(Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) b=390b = 390 We calculate the zz value of the boundary. z=b−μσ=390−480150=−0.6\begin{aligned} z &= \frac{b-\mu}{\sigma} \\\\ &= \frac{390-480}{150} \\\\ &= -0.6 \end{aligned}

We have rephrased our problem into a standard normal probability problem, because P(B<390)=P(Z<−0.6)P(B<390) ~=~ P(Z<-0.6)
