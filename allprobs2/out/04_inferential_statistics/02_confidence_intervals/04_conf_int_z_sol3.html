First, determine z⋆z^\star such that P(|Z|<z⋆)=0.88P(|Z|<z^\star)=0.88. It can help to draw a sketch.


The entire area under the density curve is 1. Thus, we can determine the areas of the tails (using symmetry). area of each tail=1−γ2=1−0.882=0.06\text{area of each tail} = \frac{1-\gamma}{2} = \frac{1-0.88}{2} = 0.06


Determine a leftward area with boundary z⋆z^\star. P(Z<z⋆)=γ+1−γ2=0.88+0.06=0.94P(Z<z^\star)=\gamma + \frac{1-\gamma}{2}= 0.88+0.06=0.94 It should be mentioned the expression can be simplified. P(Z<z⋆)=12+γ2=0.5+0.44=0.94P(Z<z^\star)= \frac{1}{2}+\frac{\gamma}{2}= 0.5+0.44=0.94


We rephrase the puzzle. We wish to determine z⋆z^\star such that P(Z<z⋆)=0.94P(Z<z^\star)=0.94. This is easy to determine using a zz-table, qnorm in R, norm.inv in spreadsheets, or other methods for evaluating the quantile function.

P(Z<1.55)=0.94P(Z<1.55)=0.94 In the original phrasing,

P(|Z|<1.55)=0.88P(|Z|<1.55)=0.88

So, z⋆=1.55z^\star = 1.55. Now, use the given expressions.

Lower bound=25.1−(1.55)(7.6)/179=24.2\text{Lower bound} = 25.1-(1.55)(7.6)/\sqrt{179} ~=~ 24.2 Upper bound=25.1+(1.55)(7.6)/179=26\text{Upper bound} = 25.1+(1.55)(7.6)/\sqrt{179} ~=~ 26

Spreadsheet

A spreadsheet can use the confidence.norm function. It takes three arguments:

  • Alpha (α\alpha) is the two-tail area, α=1−γ=1−0.88=0.12\alpha=1-\gamma=1-0.88=0.12
  • Population standard deviation, σ=7.6\sigma=7.6
  • Sample size, n=179n=179

The function returns the margin of error. margin of error=z⋆σn\text{margin of error} = \frac{z^\star \sigma}{\sqrt{n}}

So, in this case, if in a spreadsheet you typed =confidence.norm(1-0.88,7.6,179), the result would be the margin of error, 0.8831902. You then add/subtract the margin of error to/from the sample mean. x‾−ME=25.1−0.883=24.217\bar{x}-\text{ME} = 25.1-0.883 = 24.217 x‾+ME=25.1+0.883=25.983\bar{x}+\text{ME} = 25.1+0.883 = 25.983

You can do this with a spreadsheet.


R

xbar = 25.1
sigma = 7.6
n = 179
gamma = 0.88
zstar = qnorm(gamma+(1-gamma)/2)
ME = zstar*sigma/sqrt(n)
LB = xbar-ME
UB = xbar+ME
cat(sprintf("The lower bound: %.4f\nThe upper bound: %.4f",LB,UB))
